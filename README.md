# CodexGlyph Engine
A Python-based natural language processing (NLP) engine for analyzing text through semantic, morphological, and structural glyph patterns.

CodexGlyph is an experimental linguistic tool designed to explore deeper relationships within language.  
It blends modern NLP techniques with custom pattern-detection algorithms to identify glyph structures, semantic weight, conceptual categories, and morphological signatures inside raw text.
